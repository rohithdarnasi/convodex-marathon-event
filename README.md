# ConvoDex - AI Website Chatbot Generator

Built for Google Cloud Build & Blog Marathon 2025

## Overview
ConvoDex is an AI-powered chatbot generator that creates custom chatbots for any website. Simply provide a URL, and ConvoDex will crawl, process, and create an embeddable chatbot widget powered by Google Gemini.

## Technologies
- Google Cloud Run
- Google Firestore
- Google Gemini API
- Python Flask
- Beautiful Soup
- HTML/JavaScript

## Features
- Automatic website crawling
- Semantic search using embeddings
- Context-aware responses
- Easy embed widget
- Powered by Google Gemini

## Architecture
1. User submits website URL
2. Backend crawls and extracts content
3. Gemini generates embeddings
4. Stores in Firestore
5. Generates embed code
6. Chatbot answers using semantic search
